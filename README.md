# MCST

## Task 1

Скомпилировать программу из корня проекта можно следующим образом:

```bash
cd task1
make
```

Проверить программу можно следующим образом:

```bash
echo <array> | ./bin <threads_count>
```

Пример проверки:

```bash
# echo 3 75 6 2 7 8 1 9 88 | ./bin 9
>1 2 3 6 7 8 9 75 88
```

## Task 2

1. Скомпилировать программу из каталога task2:

```bash
make
```

2. Создать FIFO:

```bash
mkfifo chat.fifo
```

3. Выполнить следующую команду в двух терминалах:

```bash
./bin chat.fifo
```

Замечание: при тестировании обнаружено, что канал не является надёжным, возможна потеря сообщений по причине, не связанной с логикой программы. Проблему возможно решить, используя два FIFO-канала.
