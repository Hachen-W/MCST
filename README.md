## Task 1

Скомпилировать программу из корня проекта можно следующим образом:

```bash
cd task1
make
```

Проверить программу можно следующим образом:

```bash
echo <array> | ./bin <threads_count>
```

Пример проверки:

```bash
# echo 3 75 6 2 7 8 1 9 88 | ./bin 9
>1 2 3 6 7 8 9 75 88
```

## Task 2

1. Скомпилировать программу из каталога task2:

```bash
make
```

2. Создать FIFO:

```bash
mkfifo chat.fifo
```

3. Выполнить следующую команду в двух терминалах:

```bash
./bin chat.fifo
```

**Замечание:** при тестировании обнаружено, что канал не является надёжным, возможна потеря сообщений по причине, не связанной с логикой программы. Проблему возможно решить, используя два FIFO-канала.

## Task 9

v1 ищет строго возрастающую подпоследовательность с максимальной суммой элементов в ней. Если таких несколько, то возвращает подпоследовательность максимальной длины.

**Замечание:** программа в данном варианте обнуляет переменные на отрицательных числах, так как в подпоследовательности с максимальной суммой элементов не может быть отрицательных чисел, так как она ещё должна быть возрастающей.

v2 ищет строго возрастающую подпоследовательность максимальной длины. Если таких несколько, то возвращает подпоследовательность с наибольшей суммой элементов.

**Запуск из task9:**
```bash
python3 main.py
```

**Пример запуска (v1.py):**

