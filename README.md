## Task 1

Скомпилировать программу из корня проекта можно следующим образом:

```bash
cd task1
make
```

Проверить программу можно следующим образом:

```bash
echo <array> | ./bin <threads_count>
```

Пример проверки:

```bash
# echo 3 75 6 2 7 8 1 9 88 | ./bin 9
>1 2 3 6 7 8 9 75 88
```
